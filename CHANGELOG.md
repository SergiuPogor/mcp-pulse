All notable changes to mcp-pulse are documented here.

## [1.0.0] — 2026-03-21

### Added
- Real-time WebSocket dashboard with live call tracing
- SSE proxy for any MCP server transport
- In-memory metrics store with configurable retention
- Latency histograms (P50, P95, P99)
- Tool usage leaderboard
- Alert engine with webhook delivery
- CSV + JSON export
- CLI: `dashboard`, `proxy`, `proxy:stdio`, `alert`, `export`
- TypeScript types for all MCP protocol methods
- Dark OLED-friendly dashboard UI
- GitHub Actions CI/CD pipeline
- Docker support
